# Dashboard Search & Conversão 2025

Dashboard em Streamlit para análise mensal de:
- Users
- Searches
- Receita
- AOV
- CTR
- Conversion Rate
- No Results
- No Clicks
- Add To Cart
- Purchase Rate

## Como rodar localmente

1. Instale as dependências:

pip install -r requirements.txt

2. Execute:

streamlit run app.py

3. Faça upload do Excel consolidado (com abas "Dados" e "MoM")

---

## Deploy no Streamlit Cloud

1. Suba este repositório no GitHub
2. Acesse https://streamlit.io/cloud
3. Conecte seu GitHub
4. Selecione o repositório
5. Deploy

Pronto.
